# bash-system-info
A simple Bash script to display Linux system information.

# Bash System Information
دو زبانه
---

## English

A simple Bash script that displays useful Linux system information in a clean and organized format.

### Features

- Hostname
- Current User
- Operating System
- Kernel Version
- System Architecture
- System Uptime
- Current Date and Time
- Current Shell
- CPU Model
- CPU Core Count
- Total RAM
- Used RAM
- Free RAM
- Disk Information
- IP Address
- Default Gateway
- DNS Server

### Requirements

The script uses standard Linux utilities and does not require any additional packages.

### Installation

Clone the repository:

```bash
git clone https://github.com/alibabaali2020-ui/bash-system-info.git
```

Enter the project directory:

```bash
cd bash-system-info
```

Make the script executable:

```bash
chmod +x system-info.sh
```

Run the script:

```bash
./system-info.sh
```

### Example Output

```text
==============================
SYSTEM INFORMATION
==============================

Hostname         : ubuntu
Current User     : ali
Operating System : Ubuntu 24.04.2 LTS
Kernel Version   : 6.8.0
Architecture     : x86_64
...
```
